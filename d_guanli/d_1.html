查看系统信息

查看系统信息

查看系统信息，命令如下：

$ cat /etc/tencentos-release
或
$ cat /etc/os-release
或使用t命令
# t -s

结果如下：

[root@TENCENT64 ~]# cat /etc/tencentos-release
TencentOS Server 3.1 (TK4)
[root@TENCENT64 ~]# cat /etc/os-release

查看系统相关的资源信息

查看CPU信息，命令如下：

$ lscpu

查看内存信息，命令如下：

$ free

查看磁盘信息，命令如下：

$ fdisk -l

查看系统资源实时信息，命令如下：

$ top
