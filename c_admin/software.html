软件安装

TS安装软件指导

1. 默认源

TS2 默认安装有tlinux-release，epel-release，其分别包含tlinux.repo和epel.repo
TS3 默认安装有tencentos-release, epel-release, 其分别包含TencentOS.repo, TencentOS-Sources.repo和epel-repo, epel-modular.repo 默认使用腾讯软件源(mirros.tencent.com) 作为yum源

2. docker-ce

TS2 首先执行yum install tlinux-release-docker-ce
TS3 首先执行yum install tencentos-release-docker-ce 即可安装docker-ce的源tlinux-docker-ce.repo，会默认使用mirrors.tencent.com/docker-ce 目录 然后就可以安装docker-ce了

yum install tlinux-release-docker-ce
yum install docker-ce

若遇到冲突，需要删除旧版本

yum remove docker docker-client docker-engine docker-client-latest docker-latest

安装完docker启动遇到如下错误: Error starting daemon:Error initializing network controller: list bridge addresses failed: no available network 可通过如下命令解决:

sudo ip link add name docker0 type bridge
sudo ip addr add dev docker0 172.17.42.1/16

3. gcc update（TS2安装gcc-7.3版本）

TS2 首先执行yum install tlinux-release-gcc-update即可安装gcc updates的源tlinux-gcc-update.repo ，然后就可以安装高版本gcc了

yum install tlinux-release-gcc-update
yum remove tlinux12-compat compat-gcc-44-c++
yum update gcc

4. SCL （高版本软件）

TS2可以通过使用centos scl源的方式安装使用高版本软件，首先通过yum安装tlinux-release-scl包即可安装scl的源tlinux-sclo.repo，然后可以安装使用高版本软件仓库,以gcc为例：

yum install tlinux-release-scl
scl --list
yum install devtoolset-8-gcc
yum install devtoolset-8-gcc-c++
scl enable devtoolset-8 bash

安装路径在 /opt/rh

5. openresty

首先安装tlinux-release-openresty，安装源文件tlinux-openresty.repo（http://mirrors.cloud.tencent.com/openresty/centos/7/x86_64/ ），然后通过yum安装openresty

yum install tlinux-release-openresty
yum install openresty

6. pypi

首先通过yum安装python-pip，然后即可使用腾讯软件源加速pip

yum install python-pip  或 python2-pip 或 python3-pip

# cat /etc/pip.conf
[global]
index-url = https://mirrors.tencent.com/pypi/simple

7. ceph-fuse

首先通过yum安装tlinux-release-ceph，安装源文件tlinux-ceph.repo（http://mirrors.tencent.com/ceph/rpm-jewel/el7/ ）然后即可使用腾讯软件源安装ceph-fuse

yum install tlinux-release-ceph
yum install ceph-fuse

8. PostgreSQL

首先通过yum安装tlinux-release-pgdg，安装源文件tlinux-pgdg.repo（https://mirrors.tencent.com/postgresql/repos/yum/ ）然后即可使用腾讯软件源安装postgresql

yum install tlinux-release-pgdg
yum install postgresql12

9. Kubernetes

首先新建/etc/yum.repos.d/kubernetes.repo，其内容如下

cat << EOF > /etc/yum.repos.d/kubernetes.repo
[kubernetes]
name=Kubernetes
baseurl=https://mirrors.tencent.com/kubernetes/yum/repos/kubernetes-el7-x86_64
enabled=1
gpgcheck=0
EOF

然后就可以安装k8s了：

yum clean all
yum makecache
yum -y install kubelet kubeadm kubectl kubernetes-cni
systemctl enable kubelet && systemctl start kubelet

10. Glusterfs-7

TS2首先通过yum安装tlinux-release-gluster7 TS3请yum install tencentos-release-gluster7,安装源文件tlinux-Gluster-7.repo（http://mirrors.tencent.com/centos/7/storage/ ）然后即可使用腾讯软件源安装glusterfs-server

yum install tlinux-release-gluster7
yum install glusterfs-server

11. git-2.19

通过tlinux-testing源仓库可安装高版本软件包，比如git 2.19版本，适用于TS2 for x86_64 和 arm64：

yum --enablerepo=tlinux-testing install git
